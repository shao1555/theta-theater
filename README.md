THETA Theater
=============

getting start
-------------

    bundle install
    rackup

- rackup すると Twitter の各種トークン類の設定がはじまります。
- 起動後、 http://localhost:9292/theater を開くと、直近で投稿された全天球画像が表示される画面になります。
